In this tutorial, you learn how to write, execute and debug a proper shell program in Linux/Unix with the help of Vim editor. 
You build a birthday reminder application that triggers a mail one day before to the organizer of a specific team/project.

## What You Will Learn:

- What is Shell? And what are the different types of it?

- How to create and execute shell script using terminal/Vim editor.

- How to start from one of the shell script program (Arithemtic)

- How to properly modularize shell program

- How to pass arguments to shell program and across functions

- How to debug shell program

- How to use crontab

- How to use Vim editor effectively


## Requirements

- To complete this workshop, all you need is a Linux terminal and a connection to the Internet.
- A working knowledge of Linux/Unix is assumed, but you don't need to be a Linux/Unix guru.
